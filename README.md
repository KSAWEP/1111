# شركة العواد للتكييف

موقع ذو صفحة واحدة بنظام Page Piling - 4 صفحات مكدسة تنزل بحركة سلسة عند السكرول.

## كيف يشتغل التنقل

- **السكرول لتحت** ينقلك للصفحة التالية بحركة `translateY` (الصفحة الحالية تطلع لفوق والجديدة تجي من تحت)
- **السكرول لفوق** يرجعك للصفحة السابقة
- **النقاط الجانبية** يسار الشاشة - الضغط على أي نقطة ينقلك مباشرة
- **الكيبورد** - أسهم لفوق/تحت
- **اللمس** - السحب يشتغل على الموبايل

## الصفحات الـ 4

1. الرئيسية (Hero)
2. عن العواد
3. خدماتنا (سلايدر)
4. التزاماتنا

## التشغيل

افتح `index.html` في المتصفح مباشرة.
