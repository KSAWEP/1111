# شركة العواد للتكييف - النسخة المحدثة

## التعديلات في هذه النسخة

### 1. الألوان الجديدة
- **كحلي:** `#0a2647` (الخلفية الأساسية)
- **أزرق فاتح:** `#5BC0EB` (اللون المميز)
- **أبيض:** للنصوص
- بدلاً من الأزرق التركوازي والداكن السابق

### 2. حركة النزول الأنعم
- نفس السرعة (0.9 ثانية)
- منحنى حركة محسّن: `cubic-bezier(0.32, 0.72, 0, 1)` (أنعم وأكثر طبيعية)
- إضافة تلاشي بسيط للصفحات في الخلفية

### 3. تحسينات الجوال الشاملة

**القوائم:**
- قائمة منبثقة احترافية مع أيقونات لكل قسم
- زر هامبرغر أكبر وأوضح (44×44px)
- شعار الشركة في رأس القائمة
- أزرار "اتصل" و "واتساب" أسفل القائمة

**الأزرار والنصوص:**
- جميع الأزرار صارت أكبر وأوضح
- في الموبايل، أزرار CTA تصير full-width
- النصوص بحجم مناسب للقراءة

**السكرول:**
- خدمات تنزل بسكرول داخلي على الموبايل (لأنها 9 خدمات)
- المؤشرات الجانبية مخفية في الجوال (تأخذ مكان)
- زر "Scroll Down" مخفي في الجوال
- viewport-fit=cover للشاشات الحديثة

**Layout:**
- جميع الـ Grids تتحول لعمود واحد
- الصور بأحجام مناسبة للجوال
- الفورم ينظم رأسياً
- الفوتر يتمحور

- ✅ نظام Page Piling الأصلي

- ✅ السكرول الذكي (داخلي ثم انتقالي)
